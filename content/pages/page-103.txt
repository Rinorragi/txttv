LESSONS FROM EQUIFAX: Struts CVE-2017-5638
-------------------------------------------------

INCIDENT OVERVIEW:
March-July 2017: Attackers exploited unpatched
Apache Struts vulnerability (CVE-2017-5638) to
compromise Equifax systems.

IMPACT SCALE:
• 147 million consumers affected
• Social Security numbers exposed
• Birth dates, addresses compromised
• Credit card data for 209,000 stolen
• Settlement: $700 million

ATTACK TIMELINE:
March 7: Apache releases Struts patch
March 10: Equifax vulnerability notice
May 13: Initial breach (undetected)
July 29: Breach discovered (76 days late)
Sept 7: Public disclosure

ROOT CAUSES:
❌ Patch not applied for 2+ months
❌ Vulnerability scanner missed the issue
❌ SSL certificate expired (monitoring gap)
❌ Segmentation failures
❌ Inadequate logging

KEY TAKEAWAYS:
✓ Patch critical CVEs within 48 hours
✓ Verify scanner coverage quarterly
✓ Monitor certificate expiry
✓ Implement defense in depth
✓ Enable comprehensive audit logging
